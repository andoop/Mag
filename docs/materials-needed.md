## Asset Guidelines

- Prefer light and dark mode coverage if available.
- Avoid real API keys, private repositories, personal paths, or customer data.
- Use stable demo workspaces with realistic but non-sensitive content.
- Keep videos short: 20 to 60 seconds is enough for README/demo use.
