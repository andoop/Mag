# Code of conduct

We want **Mag** to be a welcoming project for everyone.

- **Be respectful** — Disagree on technical merits; do not harass, discriminate, or publish others’ private information.
- **Assume good intent** — Especially across languages and time zones.
- **Report problems** — Maintainer contact via GitHub (issues for public conduct topics, or private channels when appropriate).

For a fuller standard, this project endorses the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as a reference. Maintainers may remove, edit, or reject contributions (issues, comments, PRs) that violate this spirit.

---

## 中文

我们希望 **Mag** 社区保持友好与专业：对事不对人，禁止骚扰与歧视。完整行为标准可参考 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)。维护者可对不符合上述精神的 Issue、评论或 PR 进行处理。
